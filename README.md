# PlanarWM

2D scrolling window manager for the River[https://codeberg.org/river/river] compositor, implemented in Rust, based on tinyrwm[https://codeberg.org/river/tinyrwm].

## Building

```sh
cargo build --release
```

## Running

```
river -c ./target/release/planarwm
```
